## How to Use This Kit

### Step 1 — Pick the right skill file
Open the `skills/` folder and find the prompt that matches what you're building.
Each file explains what the prompt does, when to use it, and how to adapt it.

### Step 2 — Fill in the brackets
Every prompt uses `[bracket placeholders]` for the parts you customise.
Replace each one with your actual requirements before pasting into Claude.

### Step 3 — Paste into Claude
Works with:
- **Claude** (claude.ai) — recommended
- **Microsoft Copilot** (copilot.microsoft.com)
- **ChatGPT** (openai.com)

### Step 4 — Review the output critically
AI output is a starting point, not a final answer. Always:
- Check delegation limits on any gallery or collection formula
- Verify expression syntax in Power Automate before saving
- Test Adaptive Card JSON in the [Adaptive Card Designer](https://adaptivecards.io/designer/) before deploying

## How to Create Your Own Skill Files

Each `.md` file in `skills/` follows this structure — copy it to build your own:

```markdown
# Skill Name

## What this prompt does
One paragraph describing the output.

## When to use it
- Bullet list of use cases

## The Prompt
\```
[paste prompt here]
\```

## How to adapt it
- Tips for filling in the brackets

## What to expect back
Description of what Claude returns.

## Common mistakes
- Things to watch out for
```

Save your custom prompts in the `skills/` folder with a descriptive filename.
Over time this becomes your personal Power Platform prompt library.
